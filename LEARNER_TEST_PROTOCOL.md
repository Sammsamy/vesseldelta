# VesselDelta informal learner test

This is a lightweight product-usability and concept check for the Build Week submission. It is not clinical research, physician review, classroom validation, or evidence that VesselDelta improves medical competence.

## Who to ask

Invite `3–5` real learners if available: medical, nursing, PA, pharmacy, biomedical-engineering, pre-health, or physiology students. Record only a broad background label such as “second-year nursing student.” Do not collect names, health information, grades, diagnoses, or any other sensitive information.

Participation should be voluntary. Ask permission before using an anonymous quote in the submission. A teammate, the builder, or a coached participant should not be counted as independent evidence.

## Five-minute protocol

### 1. Pre-check, before showing VesselDelta

Ask the participant to choose one answer for each question without coaching:

1. In an idealized narrowed channel at the same flow drive, where should the fastest modeled flow appear?
   - A. At the narrowest throat
   - B. Only in the downstream turning region
2. If only VesselDelta’s illustrative pressure factor rises, what changes?
   - A. The separate relative wall-tension index
   - B. The CFD jet speed
3. Can the brightest model color predict vessel rupture?
   - A. Yes
   - B. No

Answer key: `A, A, B`. Record only the score out of three.

### 2. Two-minute uncoached use

Give the participant the working app and ask them to:

- inspect **Idealized artery narrowing**;
- choose **Higher pressure state**;
- open **Can this vessel rupture?**;
- complete the built-in **Three-step mechanics check**.

Do not explain the answers while they use it. Observe whether they can find the controls and note one point of confusion.

### 3. Post-check

Ask the same three questions again, in the same order. Record the score out of three and one optional answer to:

> What became clearer, if anything?

Use the quote only with explicit permission and keep it anonymous.

## Honest reporting rule

Report the result only if real participants completed the same protocol. State the sample size and call it an **informal learner test**, for example:

> In an informal uncoached test with `n = 4` health-science learners, the median three-question score changed from `2/3` before use to `3/3` after two minutes. This is early product feedback, not validated learning research.

Do not invent participants, omit negative results, call this a study, claim statistical significance, or use the result as clinical or educational validation. If fewer than three independent learners are available, use their comments only as qualitative product feedback and disclose the count.
